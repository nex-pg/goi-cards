## 説明（4000字以内）
```
語彙トレカードは、二字熟語・四字熟語・慣用句・ことわざ・用語をカードでサクサク覚えられる語彙学習アプリです。ログイン不要・アカウント作成なしで、すぐに学習を始められます。

■ 特長
・1,500語超を収録（無料で500語、Proで全語を解放）
・カードをタップしてめくる、シンプルな一問一答形式
・「わかる／わからない」で仕分けして、苦手だけを復習
・分類・難易度で出題を自由に絞り込み
・学習履歴やブックマークで復習をサポート

■ 収録カテゴリ
・二字熟語／四字熟語／慣用句／ことわざ／用語
・難易度は「易・普・難」の3段階

■ こんな方に
・中学・高校・大学受験の国語対策に
・語彙力や言語化の力、教養を伸ばしたい社会人に
・四字熟語や慣用句をすき間時間で覚えたい方に

■ Pro（買い切り）でできること
・1,500語超をすべて解放（Pro限定の単語を含む）
・一覧機能：辞書のように閲覧・検索、タグ／分類／難易度で絞り込み
・ブックマークを複数フォルダで管理
・ダークモード（表示モードの切替）
※Proは一度購入すれば追加課金なしでずっと使えます（非消耗型）。

■ ご注意
本アプリの語義・読み・分類・難易度は、学習用に簡略化した参考情報です。正確性・完全性を保証するものではありません。正式な定義は辞書等でご確認ください。
```
